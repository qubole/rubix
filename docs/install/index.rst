##################
Installation Guide
##################


This section provides instructions to install Rubix and use it with Presto or
Hive. Instructions are tested on an EMR 5.X cluster. If you want help to install 
on-prem or on other distributions, please `contact <https://groups.google.com/forum/#!forum/rubix-users>`_
the community with your questions or issues.

The instructions require `RubiX Admin <https://github.com/qubole/rubix-admin>`_
