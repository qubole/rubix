##################
Installation Guide
##################


This section provides instructions to install RubiX and use it with Presto, Hive, or Spark.
If you want help to install on-prem or on other distributions, please
`contact <https://groups.google.com/forum/#!forum/rubix-users>`_ the community with your questions or issues.

These instructions require `RubiX Admin <https://github.com/qubole/rubix-admin>`_ to be installed on your cluster.
